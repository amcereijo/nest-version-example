## Description
Project to show how to handle endpoint versioning using [Nest](https://github.com/nestjs/nest) framework (version >= 8).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:e2e

# unit and e2e tests
$ npm run test

# test coverage
$ npm run test:cov
```
